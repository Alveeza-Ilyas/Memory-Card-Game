# 🧩 Memory Card Game

A fun, interactive, and fully responsive **Memory Card Game** built using plain semantic HTML5, CSS3 Grid layouts, and Vanilla JavaScript. The game challenges players to find matching pairs of fruit emojis while tracking their performance statistics in real-time.

---

## 🎮 Features

* **Dynamic Card Board:** Generates a balanced 4x4 grid board populated with random positions of 8 distinct fruit emoji pairs.
* **Real-time Stat Tracking:** Logs and updates your total number of **Moves** and successfully completed **Matches** on the fly[cite: 37, 38].
* **Interactive Animations:** Features smooth hover scale effects and colorful status feedback when cards are flipped or correctly matched[cite: 39].
* **Win Declaration:** Automatically displays a sleek success badge overlay once all matching pairs are completely uncovered[cite: 37, 38].
* **Instant Reset:** Includes a dedicated "New Game" controller button to instantly shuffle and redeploy a fresh card layout at any moment[cite: 37, 38].

---

## 🛠️ Technology Stack

* **Structure:** Semantic HTML5 (`Memory card game.html`)
* **Styling:** CSS3 Featuring Linear Gradients, CSS Grid Layouts, and Dynamic Transform Transitions (`style.css`)[cite: 39]
* **Logic Engine:** Vanilla JavaScript DOM Manipulation, Array Shuffling algorithms, and asynchronous timeout states (`script_2.js`)
